<h4>Desenvolvimento de uma Tela de Login funcional utilizando bootstrap com Botão de FAQ interativo. </br>
Projeto criado na aula da disciplina de programação front-end, na Unicesumar, campus de Londrina.
Durante as aulas foi apresentado a nós o conceito de framework e suas funcionalidades e os comandos basicos de git como: git add, git status, git commit e git push. </br>
O botão FAQ foi criado com auxilio da IA.

</h4>

### Funcionalidades:

- Tela de login limpa e funcional.
- Botão de FAQ que não atrapalha o formulário.
- Botão Lembre-me para salvar informações.

### Ferramentas utilizadas:

- Botão FAQ: Fiz uma bolinha flutuante no canto que abre um Modal (aquela janelinha que pula na tela) com as dúvidas.
- Git: Controle de versão básico pra subir tudo aqui pro GitHub.
- Bootstrap 5: Pra não ter que fazer tudo no CSS na mão e deixar o layout responsivo.
- CSS: para estilização da pagina.
